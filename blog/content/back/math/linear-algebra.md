## Sets.
**Set** is an unordered collection of objects, for example:  
$$ \\{♠,♥,♦,♣\\} $$

$\boldsymbol \in$ - this symbol means that an object belongs to a set(equivalently, that set contains an object). For example:  
$$ ♥ \in \\{ ♠,♥,♦,♣ \\} $$
